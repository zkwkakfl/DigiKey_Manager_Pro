# 디지키 파트넘버 조회 프로그램 - 설치 및 실행 가이드

## 방법 1: 배치 파일로 실행 (간단한 방법)

### 필요 조건
- Python 3.x 설치 필요
- 인터넷 연결 (최초 패키지 설치 시)

### 실행 방법
1. `run.bat` 파일을 더블클릭합니다.
2. 최초 실행 시 필수 패키지가 자동으로 설치됩니다.
3. 프로그램이 실행됩니다.

### 장점
- 설정이 간단합니다.
- 소스 코드를 수정하고 바로 테스트할 수 있습니다.

### 단점
- Python이 설치되어 있어야 합니다.
- 다른 컴퓨터에서 사용하려면 Python을 설치해야 합니다.

---

## 방법 2: 실행 파일로 패키징 (배포용)

### 필요 조건
- Python 3.x 설치 필요 (패키징 시에만)
- PyInstaller 패키지

### 패키징 방법
1. `build_exe.bat` 파일을 더블클릭합니다.
2. PyInstaller가 자동으로 설치됩니다 (없는 경우).
3. 실행 파일이 `dist` 폴더에 생성됩니다.

### 실행 파일 사용 방법
1. `dist\DigiKey_Manager_Pro.exe` 파일을 더블클릭합니다.
2. Python 설치 없이도 실행됩니다!

### 장점
- Python 설치가 필요 없습니다.
- 다른 컴퓨터에서 바로 사용할 수 있습니다.
- 실행 파일 하나로 배포 가능합니다.

### 단점
- 패키징 시간이 걸립니다 (1-2분).
- 실행 파일 크기가 큽니다 (약 50-100MB).
- 소스 코드 수정 후 다시 패키징해야 합니다.

---

## 파일 설명

### `run.bat`
- 프로그램을 실행하는 배치 파일
- Python 설치 확인 및 필수 패키지 자동 설치
- `python main.py` 실행

### `build_exe.bat`
- 실행 파일(.exe)을 생성하는 배치 파일
- PyInstaller를 사용하여 패키징
- `dist\DigiKey_Manager_Pro.exe` 생성

---

## 문제 해결

### "Python이 설치되어 있지 않습니다" 오류
- [Python 공식 사이트](https://www.python.org/downloads/)에서 Python을 설치하세요.
- 설치 시 "Add Python to PATH" 옵션을 체크하세요.

### 패키지 설치 오류
- 인터넷 연결을 확인하세요.
- 관리자 권한으로 실행해보세요.
- `pip install -r requirements.txt` 명령을 수동으로 실행해보세요.

### 실행 파일 생성 실패
- PyInstaller가 제대로 설치되었는지 확인하세요.
- `pip install --upgrade pyinstaller` 명령으로 업그레이드해보세요.
- 관리자 권한으로 실행해보세요.

→ `run.bat` 사용 (방법 1)
